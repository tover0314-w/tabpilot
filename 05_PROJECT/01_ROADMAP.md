# Roadmap

## Phase 0：Prototype

目标：证明点击后可以自动整理顶部 tab bar。

范围：

- MV3 extension。
- Action click。
- Collect tabs。
- Rule-based grouping。
- Native tab groups。
- Basic sidebar result。

## Phase 1：MVP Aha

目标：形成首次惊喜体验。

范围：

- One-click organize。
- AI + rules classification。
- Safe duplicate cleanup。
- Undo/Restore。
- Sidebar result。
- Chat refine basic。
- Current tab summary。
- Dashboard V0。

## Phase 2：Paid Beta

目标：验证付费。

范围：

- Account。
- Hosted AI gateway。
- Usage metering。
- Dashboard Pro。
- Saved workspaces。
- Group summary。
- Multi-tab chat。
- Rules & Memory。

## Phase 3：Retention

目标：提高长期留存。

范围：

- Workspace history。
- Cloud sync。
- Advanced rules。
- Auto-add new tabs。
- Tab Knowledge。
- Export。

## Phase 4：Scale

目标：扩大市场。

范围：

- Team workspace。
- Shared templates。
- Browser history agent。
- Deep research mode。
- Cross-browser support。

## Milestones

### M1：Local Prototype

用户可点击 icon 触发本地规则分组。

### M2：AI Classification Beta

AI 能生成较好 group names 和 assignments。

### M3：Safe Dedupe Beta

安全去重可自动关闭 exact/tracking duplicates，并支持 restore。

### M4：Sidebar Agent Beta

用户可通过 sidebar 查看结果、Undo、当前页总结、聊天重分。

### M5：Dashboard V0

用户可查看 Smart Groups，调整并 apply 回浏览器。

### M6：Private Beta

邀请 20–50 个重度 tab 用户测试。

### M7：Public Launch

Product Hunt / Hacker News / X 上线。
