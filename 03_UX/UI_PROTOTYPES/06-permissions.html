05 · INCOGNITO
05Incognito — auto skipside panel stays empty
🕶

Incognito is on — TabMosaic is paused

3 incognito tabs · 0 normal tabs in this window

By default, TabMosaic does not read, summarize, or organize incognito tabs. This window's side panel is empty.

To allow TabMosaic in this incognito window: open chrome://extensions → TabMosaic → "Allow in Incognito". This setting is per-extension and remembered.
  • Incognito tabs are never sent to any AI provider — not even CLOUD mode
  • Incognito tabs are never included in Duplicates, Rules, or Auto-classify
  • When you close this incognito window, the side panel state is discarded
