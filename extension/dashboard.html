Permissions & Data Use

TabMosaic uses a small permission set for one-click grouping, undo, restore, dashboard state, and user-triggered current-page summary.

tabs Reads tab title, hostname, path, and state so it can organize all normal windows and restore safe duplicates. Required
tabGroups Creates and updates real Chrome native tab groups. Required
sidePanel Opens the sidebar control center after the toolbar click. Required
storage Stores local settings, rules, Undo/Restore snapshots, and the optional local API key. Local only
scripting + activeTab Reads visible text only from the active page after you click Summarize Current Tab. User triggered
https://api.deepseek.com/* Used only when you enable DeepSeek classification and save your own API key. Optional

Not requested: all URLs, history, bookmarks, cookies, webRequest, browsingData, or incognito access.

Beta Diagnostics

Copies a redacted local QA snapshot and recent local error summary for bug reports. It excludes URLs, hostnames, tab titles, page text, and API keys.

Local Data

Clears local rules, API key, recent results, Undo/Restore snapshots, chat drafts, and first-run privacy acceptance.
